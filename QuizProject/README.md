# 最终实现

1. 用Router实现页面跳转
2. 利用Redux进行状态管理
3. 使用蚂蚁金服的UI框架
4. 使用axios向数据库服务器发送请求
5. 分析返回ajax文件生成答题环节
   1. 在答题时，点击正确选项显示绿色，错误选项显示红色，回答错误后会显示正确选项
   2. 答题结束两秒后自动跳转下一题
   3. 10题答题结束后统计分值
